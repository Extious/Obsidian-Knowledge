# windows与linux双系统时间不同步
在对应linux中输入以下命令：
```bash
sudo apt-get install ntpdate					# 在Ubuntu下更新本地时间
sudo ntpdate time.windows.com
sudo hwclock --localtime --systohc			# 将本地时间更新到硬件上
```
